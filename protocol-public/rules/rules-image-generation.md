# Image Generation Guidelines Template

> **Note:** This file has been converted to a template. Implement with your own specific details while maintaining this structure.

## Purpose

This file provides a framework for generating consistent image prompts within the protocol. When the implementation AI doesn't have native image generation capabilities, these guidelines help create effective prompts for external image generation tools.

## Core Instructions

When the user requests an image, generate a prompt following these guidelines:

1. Consider the current conversation context
2. Format the prompt according to the target image generation platform
3. Include necessary parameters for the desired output
4. Present the prompt directly in the conversation
5. Do not edit this file when generating prompts

## Platform-Specific Formatting

### Example: Midjourney Format Template

```
[Detailed subject description] [setting/environment] [lighting] [perspective] 
[style reference] [mood/atmosphere] [color palette] [special details] 
[composition notes] [quality parameters] [aspect ratio]
```

### Platform Parameters Guide

Different platforms require different parameters. Some common examples:

- **Aspect Ratio**: --ar 16:9, --ar 1:1, --ar 3:2
- **Style Options**: --style raw, --style scenic
- **Quality Settings**: --q 2, --q 5
- **Other**: [Platform-specific parameters]

## Best Practices

1. **Be Specific**: Provide clear, detailed descriptions
2. **Reference Styles**: Mention specific artistic styles when relevant
3. **Consider Context**: Match the image to the current conversation
4. **Use Elements That Render Well**: Focus on subjects that generate consistently
5. **Follow Platform Updates**: Parameters may change with platform versions
