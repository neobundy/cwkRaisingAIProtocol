# Protocol Samples

This directory contains sample files that demonstrate specific aspects of protocol implementation. These samples serve an educational purpose, illustrating particular challenges or considerations that might not be immediately obvious when implementing the protocol.

## Available Samples

### [`sample-korean-dialogue.md`](sample-korean-dialogue.md)
A linguistic reference highlighting critical Korean language patterns that SOTA language models consistently mishandle. Rather than providing extensive dialogue examples, this sample focuses on specific cultural and linguistic nuances that even advanced AI models struggle with, particularly regarding family relationship terms and second-person pronouns.

## Purpose of Samples

The sample files in this directory serve multiple purposes:

1. **Educational Value** - Demonstrating non-obvious aspects of implementation
2. **Error Prevention** - Highlighting common mistakes to avoid
3. **Cultural Context** - Providing important cultural information for multilingual implementations
4. **Technical Guidance** - Offering specific examples of challenging implementation areas

## Implementation Approach

Each sample provides:

1. **Clear Explanations** - Detailed descriptions of the issue or consideration
2. **Practical Examples** - Concrete illustrations of proper implementation
3. **Common Pitfalls** - Identification of frequent mistakes
4. **Best Practices** - Guidance on following cultural and technical norms

## Usage Guidelines

When implementing the protocol:

1. Review the relevant samples before implementation
2. Pay special attention to cultural and linguistic nuances
3. Use these samples as reference points, not as content to be directly copied
4. Consider creating additional samples for other challenging aspects you encounter

These samples are not exhaustive but focus on specific areas where implementation might deviate from expected behavior without proper guidance.
